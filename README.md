# 🍁 楓星 EXP 效率分析器

**MapleStory Worlds 練功效率追蹤工具**

一款專為 MapleStory Worlds（楓星）設計的練功效率分析工具。透過螢幕擷取技術，自動偵測遊戲畫面中的 EXP 進度條，即時計算練功速度並預測升等所需時間。完全免費、不需安裝，用 Chrome 開啟即可使用。

v0.4（最新）

✨ 新增懸浮視窗（Picture-in-Picture） — 單螢幕也能邊打怪邊看數據，浮動顯示等級、進度、EXP/10分、距離升等時間

## ✨ 功能

- **自動偵測 EXP** — 每 0.5 秒擷取一次進度條，自動計算當前百分比與實際 EXP 值
- **等級自動辨識** — 每 60 秒自動偵測角色等級，升等後無縫銜接
- **多時段統計** — 即時顯示 1分、10分、60分及全程累積 EXP
- **升等預估** — 根據目前速率預測距離升等還需多少時間
- **跨等級累積** — 升等後數據不中斷，完整記錄整段練功歷程
- **雙版本支援** — 台服（Big Bang 前）與韓服各一個版本
- **懸浮視窗** — 單螢幕也能使用，浮動小視窗永遠置頂顯示關鍵數據

## 🚀 使用方式

1. 下載 `.html` 檔案（台服或韓服版本）
2. 用 **Chrome 瀏覽器**開啟
3. 輸入角色等級（之後會自動偵測）
4. 點「開始擷取螢幕」並選擇遊戲視窗
5. 開始練功，數據即時更新！
6. 單螢幕用戶可點「⧉ 懸浮視窗」讓數據浮在遊戲上方

## ⚠️ 注意事項

- 僅支援 **Chrome 瀏覽器**
- 預設針對 **1920×1080** 解析度校正，其他解析度可能有誤差
- 所有運算皆在**本地瀏覽器**完成，不上傳任何資料
