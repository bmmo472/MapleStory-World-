HOW TO USE

3 分鐘上手

1

下載 HTML 檔案

選擇你的伺服器版本下載（台服或韓服），或直接點「線上開啟」。

2

用 Chrome 或 Edge 開啟

工具使用 Chromium 專屬 API，Firefox 和 Safari 無法使用。

3

輸入角色等級

第一次使用時請手動輸入角色等級，之後升等會自動偵測。

4

開始擷取螢幕

點「開始擷取螢幕」並選擇你的楓星遊戲視窗，數據會即時更新。

5

開啟懸浮視窗（可選）

單螢幕玩家可以開啟懸浮視窗，讓數據一直浮在遊戲上方。
